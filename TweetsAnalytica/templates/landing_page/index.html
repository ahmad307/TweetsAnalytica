We Are Tweets Analytica

TweetsAnalytica is a web-based application built with Django framework, that applies sentiment analysis on a user's Twitter profile and visualizes analysis result.
Tweets Analytica uses Twitter API, ParallelDots API, Tweepy and Matplotlib Python libraries.
